# 학생 조회
학생 목록을 빠르게 봅니다.

## 학생 검색

상단 검색창에 검색어를 입력하고 엔터 키를 누릅니다.

### 검색법

+ 1자리 숫자: 해당 학년에 속한 학생을 모두 검색합니다.
+ 2자리 숫자: 해당 학년과 반에 속한 학생을 모두 검색합니다.
+ 4자리 숫자: 학번으로 검색합니다.
+ 1~3자리 이름: 이름으로 검색합니다.
+ `*`: 모든 학생을 검색합니다.

예를 들어, `1`은 모든 1학년, `23`은 2학년 3반, `3101`은 3학년 1반 1번을 검색합니다.

AND 연산은 쉼표, OR 연산은 띄어쓰기로 구분합니다.
예를 들어, `12 3101`은 1학년 2반에 속해 있거나 학번이 3101인 학생을 검색합니다.

### 정렬

검색 결과를 정렬하기 위해서는 검색창 오른쪽의 버튼을 누르고 정렬 기준을 선택합니다.

## 학생 정보 보기

어떤 학생의 정보를 자세히 보려면 해당 열을 클릭합니다.

## 내보내기

검색 결과를 .xlsx 파일로 내보내려면 하단의 내보내기 버튼을 클릭합니다.
